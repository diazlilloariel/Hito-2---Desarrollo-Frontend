# Ferretex Frontend (Demo)

Frontend tipo e-commerce para ferretería con flujo de compra (cliente) y panel interno (staff/manager).  
Proyecto orientado a evaluación y portafolio: UX clara, rutas limpias, estado persistido y arquitectura lista para backend.

## Stack
- React + Vite
- React Router
- Material UI (MUI)
- Persistencia local: localStorage (mock)

## Instalación y ejecución
```bash
npm install
npm run dev
